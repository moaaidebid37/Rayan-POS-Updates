# Restro POS System

نظام Point of Sale (POS) كامل لمطعم فرع واحد مبني باستخدام HTML, CSS, و JavaScript.

## المميزات

### الصفحات الرئيسية:
1. **الصفحة الرئيسية (Dashboard)** - إحصائيات سريعة والطلبات النشطة
2. **صفحة الكاشير (Cashier)** - أخذ الطلبات مع دعم Dine-in, Takeaway, و Delivery
3. **إدارة الطاولات (Tables)** - عرض وإدارة جميع الطاولات مع حالاتها
4. **الطلبات (Orders)** - تاريخ الطلبات، الطلبات المعلقة، والطلبات الأوفلاين

6. **التقارير (Reports)** - تقارير المبيعات والإحصائيات والدرج النقدي
7. **الإعدادات (Settings)** - إعدادات الحساب والنظام
8. **إدارة القائمة (Menu)** - إضافة وتعديل وحذف الأصناف

## المميزات الوظيفية

- ✅ أخذ الطلبات (Dine-in, Takeaway, Delivery)
- ✅ إدارة الطاولات مع حالات مختلفة (Vacant, Occupied, On Hold, Disabled)
- ✅ سلة طلبات تفاعلية مع تعديل الكميات والخصومات
- ✅ تطبيق الخصومات والكوبونات
- ✅ حفظ الطلبات المعلقة (Hold Orders)
- ✅ تاريخ الطلبات والمبيعات

- ✅ تقارير المبيعات (يومي، أسبوعي، شهري)
- ✅ الدرج النقدي (Cash Drawer)
- ✅ طباعة الفواتير
- ✅ إدارة القائمة (إضافة/تعديل/حذف الأصناف)

## البنية

```
rayan-pos/

```

- **Inter** - خط رئيسي لجميع النصوص

## الاستخدام

1. افتح `index.html` في المتصفح
2. استخدم القائمة الجانبية للتنقل بين الصفحات
3. جميع البيانات تُحفظ محلياً في localStorage

## ملاحظات

- النظام يستخدم localStorage لحفظ البيانات (لا يتطلب خادم)
- البيانات تبقى محفوظة حتى يتم مسح بيانات المتصفح
- يمكن طباعة الفواتير مباشرة من المتصفح

## المتطلبات

- متصفح حديث يدعم ES6+ و localStorage
- لا حاجة لتثبيت أي مكتبات خارجية
